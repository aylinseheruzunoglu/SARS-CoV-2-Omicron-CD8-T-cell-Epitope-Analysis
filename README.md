# SARS-CoV-2-Omicron-CD8-T-cell-Epitope-Analysis
Python-based bioinformatics pipeline developed to analyze the impact of SARS-CoV-2 mutations on CD8+ T cell epitopes, predict HLA Class I binding affinities using NetMHCpan 4.1, and visualize epitope conservation across variants (BA.2.86, JN.1, LP.8.1, NB.1.8.1, XEC).
